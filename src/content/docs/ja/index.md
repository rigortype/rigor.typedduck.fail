参照ドキュメントは、ビルド時に `rigortype/rigor` の submodule から生成されます。英語版を先に公開し、日本語翻訳は同じパスで `src/content/docs/ja/` に追加していく構成です。

## ドキュメント構成

- **ハンドブック**: Rigor を日常的に使うための手順中心の資料です。
- **型仕様**: 型モデル、型演算子、推論規則を扱います。
- **内部仕様**: コントリビューター向けの実装上の契約をまとめます。
- **設計判断**: upstream リポジトリにある ADR と設計履歴です。
