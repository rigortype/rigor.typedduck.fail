# 日本語訳 用語棚卸し表（作業ドキュメント）

サイト全体（`src/content/docs/ja/` 141ファイル）の専門用語の表記ゆれを
洗い出し、用語集（`translation-glossary.md`）への反映方針を決めるための
作業用メモ。確定したら用語集へ移し、本ファイルの該当行は消してよい。

出現数は fenced code block / inline code を除いた本文ベースの概算。
漢語側は一般語（「事実上」「表面的」「不変」等）の混入があるため、
**同一概念を指す箇所のみが置換対象**である点に注意。

## 分類ルブリック

- **A. 漢語の定訳が確立** → 定訳を主表記に統一
- **B. 定訳はあるが難読/非主流** → カタカナを主表記、初出で `（english、◯◯とも）` を併記
- **C. Rigor固有 or 定訳なし** → カタカナ主表記、初出で英語を併記
- **D. 一般CSで定着済みカタカナ** → そのまま（触らない）

## 決定事項（2026-05-29 確定・適用済み）

1. **subtyping**: 型=`部分型` / 過程=`サブタイピング`。`サブタイプ`(カタカナ名詞)は廃止し`部分型`へ。→ **適用済み（25件）**
2. **narrowing**: `ナローイング` 据え置き。→ 変更なし
3. **gradual**: 複合語含め全面 `漸進的`。本文では元の `gradual xxx` を初出で併記。→ **適用済み（95件＋注記4件）**
4. **nominal**: `公称`系を `名前的`へ。初出の「公称型とも」注記のみ保持。→ **適用済み（55件）**
5. **新ポリシー**: 漢語・カタカナを問わず、各ページ初出で英語原語を一度併記（検索性）。→ 用語集に明記済み・**本文適用は未着手**
6. 小揺れ: `サウンドネス`→`健全性`、本文主表記の `交差型`→`インターセクション型` を統一。`制御フロー依存`は論文レビューノートの引用文脈なので据え置き。
